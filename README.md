# RTWallpaper
Un motor de fondos de pantalla animados para Windows, ligero y optimizado para juegos
Es una aplicación sencilla que permite poner videos (.mp4, .mkv) como fondo de pantalla dinámico, reemplazando el fondo estático de Windows.
Usa mpv, formatos permitidos: (.mp4), (.mkv), (.webm), (.mov), (.avi), (.wmv), (.flv), (.mpg), (.mpeg), (.gif)

# Tech Stack
Language: C#
Framework: .NET Framework 4.8 / WinForms
Video Engine: MPV
APIs: Win32 API (User32.dll, Kernel32.dll)
